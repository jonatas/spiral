# Spiral Examples & Walkthroughs

Learn how to use Spiral through these hands-on examples.

## Walkthroughs
- **[short_walkthrough.sql](short_walkthrough.sql)**: **(Recommended)** The fastest way to see the complete lifecycle (1-minute setup).
- **[basic_walkthrough.sql](basic_walkthrough.sql)**: Introduction to magic comments and basic rollups.
- **[hacker_walkthrough.sql](hacker_walkthrough.sql)**: Deep dive into the internal mapping and bitwise math.
- **[master_walkthrough.sql](master_walkthrough.sql)**: Advanced configuration and manual hierarchy management.
- **[acceleration_walkthrough.sql](acceleration_walkthrough.sql)**: Demonstrates transparent query acceleration and hierarchical slicing.

## Demos
- **[demo.sql](demo.sql)**: A polished demonstration of all major features working together.

## Usage
Run any example using pgrx:
```bash
cargo pgrx run pg16 < examples/short_walkthrough.sql
```
